# Slice Spec

## Outcome

The workspace contains a minimal but concrete enterprise harness MVP with visible governance files, visible work artifacts, local validation hooks, and a runnable Java reference service.

## Acceptance

- governance files exist and are readable
- harness directories and sample artifacts exist
- project settings reference local validation hooks
- reference-service contains a cancel-order vertical slice
- OpenAPI and controller consistency can be validated locally
- unit and integration tests can pass locally
